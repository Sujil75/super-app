## Overview

Super App is an all-in-one productivity application that combines multiple utilities into a single, responsive interface. Rather than switching between several applications, users can access commonly used tools from one dashboard.

The project is built with modern React practices and focuses on clean architecture, scalability, and an excellent user experience.

## Getting Started

### Clone the repository

```bash
git clone https://github.com/Sujil75/super-app.git
```

### Navigate into the project

```bash
cd super-app
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

---

## Environment Variables

Create a `.env` file.

```env
VITE_WEATHER_API_KEY=your_api_key
VITE_NEWS_API_KEY=your_api_key
```
